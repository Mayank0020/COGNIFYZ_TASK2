# Create a restaurant recommendation system based on user preferences.

## Overview

This repository contains a Python notebook that using feature extraction technique Recommends the restaurants based on user preferences

## Contents

The repository contains the following file:-

1.  task_2.ipynb : A jupyter notebook including the code of the Linear Regression Model and Decision Tree Regressor Model
2.  requirements.txt : A text file containing the libraries used to create this project
3.  Dataset .csv : CSV file cotaining the aggregate rating and features of multiple restaurants

## Getting Started :-

1. Clone the repository
2. Install the required python libraries from requirments.txt
3. Open the task_2.ipynb on your system and run the file

## Results :-

This file effectively tells the user the desired Restaurant based on the prompt
